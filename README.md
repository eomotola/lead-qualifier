# AI Lead Qualification Workflow
Overview
AI-powered lead qualification workflow built with Tally, Make.com, Airtable, Gmail, and Slack.

This project automates the lead qualification process for businesses using a no-code automation workflow.

Instead of manually reviewing every inquiry, the system automatically captures lead information, stores it in Airtable, sends a confirmation email to the prospect, and instantly notifies the sales team in Slack.


Problem:
Many businesses lose potential customers because lead management is slow and manual.

Sales teams often spend valuable time copying form submissions, sending emails, and notifying team members.

Solution
This workflow automates the entire lead intake process, allowing businesses to respond to prospects faster while keeping all lead information organized.

Tech Stack:
  1.  Tally
  2.  Make.com
  3.  Airtable
  4.  Gmail
  5.  Slack

Workflow:
  1. A new lead record is created in Airtable.
  2. Make.com monitors Airtable for new records.
  3. The workflow uses a Router to process multiple actions simultaneously.
  4. A personalized email is sent to the lead via Gmail.
  5. A Slack notification is sent to the sales team with the lead details.
  6. The Airtable record is updated to indicate that the automation has been completed.

Features
  1. Automated lead capture
  2. Lead database
  3. Email automation
  4. Slack notifications
  5. End-to-end workflow automation
